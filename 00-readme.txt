Allows the user to download their bookmarked works from AO3. Does not support private bookmarks.

Usage:
sh 10-download.sh -u UID [-f FORMAT] [-p PROCESSES]

Options:
-u  User ID on AO3 [required]
-f  Format to download. Valid values are epub (default), mobi, pdf, and html
-p  Processes to run at once [optional]

Happy downloading!
